# Daily Review

一天结束时做日复盘：记录进展、规划明天。

## 复盘流程

1. **今日活动**
   - 找出今天修改过的所有笔记
   - 识别新创建的笔记
   - 看看各项目的工作

2. **进展评估**
   - 完成了什么？
   - 卡住或被堵住的是什么？
   - 冒出来什么意外发现？

3. **捕获洞察**
   - 今天的关键收获
   - 发现的新连接
   - 出现的问题

4. **明天准备**
   - 前 3 个优先事项
   - 待闭环的事
   - 要探索的问题

## 输出格式

创建或更新每日笔记：

```markdown
# 日复盘 - [日期]

## 完成

- ✓ [完成项 1]
- ✓ [完成项 2]

## 推进

- [项目/Area]: [推进了什么]
- [项目/Area]: [推进了什么]

## 洞察

- [关键意识或连接]
- [重要收获]

## 卡住 / 受阻

- [没推进的事 + 为什么]

## 出现的新问题

- [冒出来的新问题]
- [要研究的事]

## 明天聚焦

1. [优先 1]
2. [优先 2]
3. [优先 3]

## 待闭环

- [ ] [要记得做的事]
- [ ] [要跟进的人]
- [ ] [要发展的想法]
```

## 附加动作

- 已完成的项目任务挪到归档
- 更新项目状态笔记
- 链接相关发现
- 标记需要关注的条目
