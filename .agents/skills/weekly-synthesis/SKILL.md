# Weekly Synthesis

对本周工作和思考做一份全面综合。

## 分析流程

1. **收集本周工作**
   - 本周创建的所有笔记
   - 本周修改的所有笔记
   - 有过活动的项目

2. **识别模式**
   - 反复出现的主题
   - 共同的挑战
   - 突破时刻
   - 能量模式（什么提神 vs 什么耗神）

3. **综合学习**
   - 浮现出来的关键洞察
   - 思考的演变
   - 发现的连接
   - 回答和提出的问题

4. **评估进展**
   - 推进的项目
   - 维护的 area
   - 新增的资源
   - 已归档的条目

## 输出格式

创建一份周综合笔记：

```markdown
# 周综合 - [日期] 这一周

## 一周概览

- 创建笔记数：[X]
- 活跃项目：[列表]
- 主要成就：[列表]

## 关键主题

### 主题 1: [名称]

- 出现的地方：[上下文]
- 为什么重要：[意义]
- 下一步：[要做什么]

### 主题 2: [名称]

- 出现的地方：[上下文]
- 为什么重要：[意义]
- 下一步：[要做什么]

## 主要洞察

1. [洞察 + 上下文]
2. [洞察 + 上下文]

## 各项目进展

### [项目名]

- 推进了：
- 卡住的：
- 下周聚焦：

## 出现的问题

- [问题 1 — 以及为什么重要]
- [问题 2 — 以及为什么重要]

## 能量审计

- 提神的：
- 耗神的：
- 要调整的：

## 建立的连接

- [笔记 A] ←→ [笔记 B]：[为什么重要]
- [概念 X] ←→ [概念 Y]：[新理解]

## 下周意图

1. [主聚焦]
2. [次聚焦]
3. [想探索的]

## 待处理

- Inbox 条目：[数量]
- 孤儿笔记：[列表]
- 缺失连接：[已识别]
```

## 后续动作

- 归档已完成项目
- 清理 inbox
- 更新项目状态
- 规划下周聚焦
