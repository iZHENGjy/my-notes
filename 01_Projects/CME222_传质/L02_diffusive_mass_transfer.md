# L02 — 扩散传质：浓度、速度、Fick 第一定律

> [!info] 这一讲在干什么
> L01 是「为什么要学传质 / 传质三种机制概览」，**L02 把扩散这种机制的"语言"打好底**——所有后续公式都用浓度（4 种）、速度（3 种）、通量（4 种）表达，一次说清楚怎么来怎么换算。

---

## 学习目标

- 看到一个混合体系，能用**浓度**描述各组分多少（4 种写法）
- 看到混合体系流动时，能区分**整体流动速度** vs **物种相对流动速度**
- 看到一个浓度梯度，能用 **Fick 第一定律**算扩散通量
- 能把"扩散通量 + 对流通量"加起来得到**总通量 N_A**（这是后面所有传质方程的起点）

---

## 知识块 1：浓度（Concentration）—— 4 种写法

混合体系里描述"组分 A 占多少"有四种方式，**互相能换算**。

![species A 在混合体中的局部分布](_attachments/CME%20222%20Lecture%202/images/CME222_L02_p04_mass-concentration-mixture_concept.jpg)

> [!tip] 配图含义
> 一个混合气泡里，●、△、○ 三种粒子代表多组分；箭头点中的"●"是 species A。**浓度的核心物理直觉**就是「单位体积里有多少 A」——质量基或摩尔基只是计量方式。

### 1.1 质量基（mass-based）

| 量 | 公式 | 物理意义 |
|---|---|---|
| 质量浓度 ρ_A | $\rho_A = m_A / V$ | 单位体积里 A 的**质量** |
| 总质量浓度 ρ | $\rho = \sum_i \rho_i$ | 整体密度 |
| 质量分数 w_A | $w_A = m_A/m = \rho_A/\rho$ | A 占总质量的比例（无量纲） |
| 比体积 υ | $\upsilon = V/m = 1/\rho$ | 单位质量占的体积 |

**约束**：$\sum_i w_i = 1$

### 1.2 摩尔基（molar-based）

| 量 | 公式 | 物理意义 |
|---|---|---|
| 摩尔浓度 c_A | $c_A = n_A/V = \rho_A/M_A$ | 单位体积里 A 的**物质的量** |
| 气相摩尔浓度（理想气体） | $c_A = P_A/(RT)$ | 用分压算（来自 PV = nRT） |
| 总摩尔浓度 c | $c = \sum_i c_i$ |  |
| 摩尔分数（液相） x_A | $x_A = n_A/n = c_A/c$ |  |
| 摩尔分数（气相） y_A | $y_A = c_A/c = P_A/P$ |  |

**约束**：$\sum_i y_i = 1$（气相）或 $\sum_i x_i = 1$（液相）

### 1.3 质量分数 ↔ 摩尔分数 换算

$$
y_A = \frac{w_A/M_A}{w_A/M_A + w_B/M_B}, \qquad
w_A = \frac{x_A M_A}{x_A M_A + x_B M_B}
$$

> [!warning] 易错点
> 算空气这类已知摩尔分数（O₂ 21%、N₂ 79%）的体系时，**先算 mean MW** $\bar M = \sum y_i M_i$，再用 $w_A = y_A M_A / \bar M$ 反推质量分数——比直接套换算公式更不容易出错。

### Example 1（空气组成 @ 25 °C, 101 300 Pa）

已知 $y_{O_2} = 0.21$、$y_{N_2} = 0.79$，$M_{O_2} = 0.032$ kg/mol、$M_{N_2} = 0.028$ kg/mol。

| 求什么 | 怎么算 | 结果 |
|---|---|---|
| (i) 平均分子量 $\bar M$ | $\bar M = 0.21 \times 0.032 + 0.79 \times 0.028$ | **0.0288 kg/mol** |
| (ii) 质量分数 | $w_{O_2} = (0.21 \times 0.032)/0.0288$ | $w_{O_2} = 0.233, w_{N_2} = 0.767$ |
| (iii) 摩尔浓度 c | $c = P/(RT) = 101300/(8.314 \times 298)$ | **40.9 mol/m³** |

## 知识块 2：速度（Velocity）—— 3 个层次

> [!important] 核心认知
> 混合体系流动时，**有 3 种"速度"在打架**：
> 1. 单个物种的速度 $v_i$（A 跑得多快、B 跑得多快）
> 2. **整体平均速度** $v$（整团流体在跑多快）
> 3. **扩散速度** $v_{i,d} = v_i - v$（物种 i 相对整体的"游离量"）
>
> Fick 第一定律算的是扩散速度产生的通量，**不是**总速度产生的通量——这就是为什么后面要做"通量分解"。

### 2.1 平均速度（mass-average vs molar-average）

| 名称 | 定义 | 何时用 |
|---|---|---|
| 质量平均速度 $v$ | $v = \sum_i \rho_i v_i / \rho = \sum_i w_i v_i$ | 动量守恒、Navier-Stokes（流体力学） |
| 摩尔平均速度 $V$ | $V = \sum_i c_i v_i / c = \sum_i x_i v_i$ | 摩尔守恒、化学反应（化工传质） |

> [!note] 为什么有两种？
> 流体力学习惯用质量加权（动量 = 质量 × 速度），化工传质习惯用摩尔加权（反应配比按摩尔走）。两种平均不等价，**但在等摩尔反向扩散等场景下数值上重合**。

### 2.2 扩散速度（diffusion velocity）

$$
v_{i,d} = v_i - v \quad \text{（质量基）}, \qquad V_{i,d} = v_i - V \quad \text{（摩尔基）}
$$

**几何含义**：站在"流体本体"上看，物种 i 相对它的速度。

### Example 2（混合体系速度计算）

给定 $w_{H_2} = 0.05$, $w_{N_2} = 0.95$；$v_{H_2} = 4$ m/s、$v_{N_2} = 0.5$ m/s。

- 质量平均：$v = 0.05 \times 4 + 0.95 \times 0.5 = 0.675$ m/s
- 扩散速度：$v_{H_2,d} = 4 - 0.675 = 3.325$ m/s

## 知识块 3：Fick 第一定律（扩散通量 J）

### 3.1 核心公式

$$
\boxed{\,J_{A,z} = -D_{AB}\frac{dc_A}{dz}\,}
$$

| 符号 | 含义 | 单位 |
|---|---|---|
| $J_{A,z}$ | A 在 z 方向的**扩散摩尔通量**（相对运动流体）| mol/(m²·s) |
| $D_{AB}$ | A 在 B 中的**互扩散系数** | m²/s |
| $dc_A/dz$ | 浓度梯度 | mol/m⁴ |

**负号的物理意义**：A 从高浓度往低浓度跑，所以梯度（$dc_A/dz$）和通量方向相反。

### 3.2 4 种等价形式（按梯度变量分）

| 通量 | 梯度 | Fick 形式 |
|---|---|---|
| $J_A$（摩尔扩散通量）| $\nabla c_A$ | $J_A = -D_{AB}\nabla c_A$ |
| $J_A$ | $\nabla y_A$（摩尔分数）| $J_A = -c D_{AB}\nabla y_A$ |
| $j_A$（质量扩散通量）| $\nabla \rho_A$ | $j_A = -D_{AB}\nabla \rho_A$ |
| $j_A$ | $\nabla w_A$（质量分数）| $j_A = -\rho D_{AB}\nabla w_A$ |

> [!tip] 用哪一个？
> 看你手上的数据是什么——给 c_A 用第一行，给 y_A 用第二行……不要硬转。

### 3.3 D_AB 与 D_BA 的关系

- **气相 / 等摩尔体系**：$D_{AB} = D_{BA}$（来自 $J_A = -J_B$ 和总通量守恒）
- **液相**：$D_{AB} \neq D_{BA}$（液相分子相互作用复杂，L04 会用 Leffler-Cullinan 处理）

### Example 3（He-N₂ 二元扩散，PPT 笔误警告）

> [!warning] PPT 笔误说明
> 原 PPT 写 "the partial pressure of He is 0.6 atm and the partial pressure of **He** is 0.2 atm at the other end"——第二个应该是同一气体 He 的另一端分压（不是 N₂）。意思就是 He 分压从 0.6 atm 降到 0.2 atm，跨越 0.2 m。

### Example 4（铁板渗碳）

| 给定 | 数值 |
|---|---|
| 温度 T | 700 °C |
| c₁（z=5 mm 处）| 100 mol/m³ |
| c₂（z=10 mm 处）| 66.67 mol/m³ |
| D（C in Fe）| $3 \times 10^{-7}$ cm²/s = $3 \times 10^{-11}$ m²/s |

$$
J = -D\frac{\Delta c}{\Delta z} = -3\times10^{-11} \cdot \frac{66.67-100}{0.005} = +2.0\times 10^{-7}\ \text{mol/(m²·s)}
$$

> [!warning] 单位坑
> D 的常见单位是 **cm²/s**（化工 / 材料界）和 **m²/s**（SI），差 $10^{-4}$ 倍——一不小心结果就差 4 个量级。**先把 D 转成和 dc/dz 配套的 SI**，再代入。

## 知识块 4：通量分解 —— 总通量 N_A 怎么来

### 4.1 速度分解（最关键的物理直觉）

![v_A 速度分解：v_A = V_Ad + V](_attachments/CME%20222%20Lecture%202/images/CME222_L02_p21_velocity-decomposition_concept.jpg)

> [!important] 这张图看懂了，整门课的通量公式都好懂
> 站在地面（静止参考系）看 A 跑：
> - **紫色 v_A** = A 相对地面的总速度
> - **红色 V_Ad** = A 相对流体本体的扩散速度（Fick 算的就是这个）
> - **绿色 V** = 流体本体相对地面的对流速度
>
> $v_A = V_{Ad} + V$ —— 两边乘 $c_A$ 得到「**总通量 = 扩散通量 + 对流通量**」。

### 4.2 三个通量的物理含义

定义 **总通量** $N_A \equiv c_A v_A$（A 相对静止参考系的摩尔通量）。两边乘 $c_A$ 后：

$$
\underbrace{c_A v_A}_{N_A,\ \text{总通量}} = \underbrace{c_A V_{Ad}}_{J_A,\ \text{扩散通量}} + \underbrace{c_A V}_{\text{对流通量}}
$$

| 项 | 通量名 | 物理意义 | 怎么算 |
|---|---|---|---|
| $c_A v_A$ | **总通量 N_A** | A 相对静止管壁穿过单位面积的速率 | 这是测得到的量 |
| $c_A V_{Ad}$ | **扩散通量 J_A** | A 因浓度梯度而扩散的部分 | Fick 第一定律 |
| $c_A V$ | **对流通量** | A 因整体流体被带着跑的部分 | $c_A V = y_A \cdot c V = y_A (N_A + N_B)$ |

### 4.3 N_A 的标准表达式（重头戏）

把 $V = (1/c)(c_A v_A + c_B v_B) = (N_A + N_B)/c$ 代回，得到：

$$
\boxed{\,N_A = \underbrace{-c D_{AB}\nabla y_A}_{\text{扩散贡献 (Fick)}} + \underbrace{y_A (N_A + N_B)}_{\text{对流贡献 (bulk)}}\,}
$$

**多组分扩展**：$N_A = -c D_{AM}\nabla y_A + y_A \sum_i N_i$

### 4.4 4 种等价的 Fick 速率方程

| 通量 | 梯度变量 | 完整公式 |
|---|---|---|
| $n_A$ 质量基 | $\nabla w_A$ | $n_A = -\rho D_{AB}\nabla w_A + w_A(n_A + n_B)$ |
| $n_A$ 质量基 | $\nabla \rho_A$ | $n_A = -D_{AB}\nabla \rho_A + w_A(n_A + n_B)$ |
| $N_A$ 摩尔基 | $\nabla y_A$ | $N_A = -c D_{AB}\nabla y_A + y_A(N_A + N_B)$ |
| $N_A$ 摩尔基 | $\nabla c_A$ | $N_A = -D_{AB}\nabla c_A + y_A(N_A + N_B)$ |

> [!tip] 类比记忆
> **染料气球丢进流动河流**：染料径向扩散（J_A，浓度梯度）同时被河流向下游带走（c_A V，对流）—— $N_A$ 是岸上观察者看到的总染料通量。

### 4.5 两种典型边界条件（后面 L05+ 会反复用）

| 场景 | 关系 | 公式 |
|---|---|---|
| **等摩尔反向扩散**（Stefan 管 / 蒸馏）| $N_A = -N_B$ | $N_A = -c D_{AB}\nabla y_A$（对流项消失） |
| **A 单向扩散过停滞 B**（吸收）| $N_B = 0$ | $N_A = -c D_{AB}\nabla y_A / (1 - y_A)$（含 ln 形式） |

## 知识块 5：Take Home Message（速查）

| # | 公式 / 概念 | 一句话 |
|---|---|---|
| 1 | $\rho_A, c_A, w_A, x_A/y_A$ | 浓度 4 种写法可互换 |
| 2 | $v, V, v_{i,d}$ | 平均速度 + 扩散速度（物种相对本体） |
| 3 | $J_A = -D_{AB} dc_A/dz$ | Fick 第一定律（扩散通量） |
| 4 | $v_A = V_{Ad} + V$ | 总速度 = 扩散速度 + 对流速度 |
| 5 | $N_A = -c D_{AB}\nabla y_A + y_A(N_A + N_B)$ | 总通量 = 扩散贡献 + 对流贡献（**所有传质问题的起点**） |
